# ChatGPT-MCP

A Model Context Protocol (MCP) integration for ChatGPT.

## 簡介

本項目提供 ChatGPT 與 MCP (Model Context Protocol) 的整合方案，讓 AI 能夠與外部工具和服務進行無縫交互。

MCP 是一個開放標準協議，用於連接 AI 助手與各種數據源和工具，實現更強大的 AI 應用場景。

## 功能特點

- 🔗 無縫連接 ChatGPT 與各種 MCP 服務
- 🛠️ 支持多種工具調用
- 📚 易於擴展和自定義
- 🚀 簡單易用的配置方式
